SOBRE EL PROYECTO

Minigames Arcade es una colección de juegos clásicos del navegador, rediseñados con una estética retro-neón moderna. El proyecto nació con el objetivo de ofrecer entretenimiento inmediato, sin descargas ni registros.

⚙️ TECNOLOGÍAS USADAS

📄HTML5 semántico
🎨CSS3 + Variables
⚡JavaScript puro
🎭Canvas API
💾localStorage
🔤Google Fonts

✨ CARACTERÍSTICAS

  • Sin dependencias externas: Vanilla JS puro, sin frameworks ni librerías.
  • Responsive: Diseño adaptable para móvil, tablet y escritorio.
  • Puntuaciones persistentes: Los récords se guardan en localStorage.
  • Accesible: Atributos ARIA y controles de teclado donde aplica.
  • Listo para AdSense: Espacios publicitarios integrados.
